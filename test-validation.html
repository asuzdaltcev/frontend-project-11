Состояние ошибки (красная рамка)

Ссылка должна быть валидным URL

Успешное состояние (зеленая рамка)

URL корректен

Обычное состояние (синяя рамка при фокусе)

Как тестировать в основном приложении:

  1. Откройте http://localhost:5173/
  2. Попробуйте ввести невалидный URL - должна появиться красная рамка
  3. Попробуйте ввести валидный URL - рамка должна стать обычной, а после успешного добавления появится зеленое сообщение
  4. Попробуйте добавить тот же URL повторно - снова должна появиться красная рамка
